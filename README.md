**Norteño Chilis — Sitio Web Oficial**

Este proyecto representa el sitio web oficial de *Norteño Chilis*, una cadena de restaurantes de comida mexicana ubicada en Durango, México. El objetivo principal del sitio es transmitir la esencia del restaurante mediante una experiencia visual elegante, moderna y cálida, alineada con la identidad auténtica y tradicional del sabor norteño.

El sitio fue construido utilizando **HTML, CSS y JavaScript**, cuidando cada detalle del diseño para que sea completamente responsivo y accesible desde cualquier dispositivo. La interfaz combina tonos claros, cafés y acentos cálidos que evocan la estética del restaurante, logrando un estilo profesional con un ambiente acogedor.  
Se eliminaron dependencias como Bootstrap para crear una estructura más limpia y un diseño totalmente personalizado.

El proyecto incluye un **slider adaptable**, un **modo oscuro persistente**, navegación móvil optimizada y múltiples páginas como *Inicio*, *Nosotros*, *Menú* y *Contacto*, todas pensadas para facilitar la consulta de información esencial por parte de los clientes. Además, se reorganizó por completo la estructura del proyecto para mantener un código más ordenado, fácil de escalar y mantener.

Este repositorio tiene como finalidad servir como la base oficial del sitio web de Norteño Chilis, permitiendo futuras mejoras, integración de nuevos contenidos, optimizaciones visuales y funcionales, así como soporte para crecimiento digital del restaurante.
